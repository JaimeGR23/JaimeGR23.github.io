Overview

This project implements a 2D fluid simulation using Smooth Particle Hydrodynamics (SPH) in C++ with Direct2D. It features a dedicated multi-threaded simulation loop, a reusable thread pool, and a pre-allocated spatial grid for neighbour queries. The simulation runs in real time (~2 ms per frame with 1000+ particles) and includes interactive behaviour: particles react to window movement as if in a glass container, and can be added or removed dynamically via input.

Key Features

Developed entirely in modern C++, focused on solving multi-threading race conditions, optimising memory usage, and enabling interactive fluid behaviour.

Interactive Window Reaction

The fluid responds to window movement, simulating the effect of shaking a glass box filled with liquid. Achieved by decoupling from the OS message loop and running simulation/render in a dedicated thread.

Thread Pool with Safe Synchronisation

A persistent thread pool reduces overhead, while an even/odd column update pass guarantees no two neighbouring cells are processed simultaneously. This avoids race conditions and keeps the simulation stable in real time.
