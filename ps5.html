Chee22 – PS5 Devkit

Custom Engine | Data-Oriented | PBR + Retro Aesthetic

Built from scratch on PlayStation 5 Devkit

Overview

Chee22 is a multiplayer local chess-like game built entirely from scratch using the PlayStation 5 Devkit and Sony’s private API. The project was developed as part of a university module at Sheffield Hallam University and received a First Class grade. We implemented a custom engine that fused multiple students’ engines into one, focusing on efficiency, polish, and compliance with Sony’s TRCs.

  • Platform: PlayStation 5 (devkit)
  • Engine: Custom (C++ modern, data-oriented design, JSON-driven)
  • Rendering: PBR implemented (not fully visible in final game due to design choices)
  • Team: Jaime Garrido, Marcos Pardo, Juan Avilés, Nacho Avilés, Jorge Alemany, Duncan Cummings (Designer)
  • Development time: 1 semester
  • Grade: First Class

Contributions

As Graphics Programmer and Gameplay Programmer, I worked on core rendering systems, performance, gameplay logic and camera behaviour. Below are some of the most representative highlights, though they only cover part of my contributions:

Turn-Based Camera System

Implemented a smooth orbital camera around the chessboard, lerping between players’ perspectives when turns change.

Procedural Postprocessing System

Developed a Python-reflective postprocessing pipeline that allowed stacking visual effects (posterisation, pixelation, etc.) in any order, giving the game its unique retro look despite lacking dedicated artists.

PBR + Retro Hybrid Rendering

Designed and implemented a PBR rendering system. While the final game focused on retro aesthetics, this screenshot showcases the underlying rendering quality with a hybrid look.

Gameplay Logic & Piece Interactions

Programmed the gameplay systems: piece movement rules, board logic, turn resolution, and small polish touches like tile animations (lifting slightly when selected) for visual feedback.
