Encestoide

Unity Basketball Game · Solo Development

Precision Mechanics · Cinemachine Cameras · Particle FX

Role: Solo Dev · Engine: Unity · Scope: Gameplay/FX/Camera · Year: 2023

  • Watch Gameplay

Overview

Encestoide is a complete prototype in Unity focused on math-driven shooting mechanics (sinusoidal timing bar + accuracy-scaled error vector), fluid movement with SLERP, Cinemachine cameras, and particle FX. Goal: a closed gameplay loop, clear feedback, and visual polish in a solo project.

Key Features

I developed all aspects of the game: gameplay systems, camera control, and user interface. The highlights below showcase the most relevant features.

Sinusoidal Precision Bar + Scaled Error Vector

Implemented a 3D sinusoidal precision bar. Releasing at centre guarantees a perfect shot; deviation injects an error vector scaled to inaccuracy.

8-Way Movement with SLERP + Force Pull FX

Direction changes smoothed via Quaternion SLERP for natural 8-way movement. Ball retrieval uses a “force pull” mechanic with particle FX.

Cinemachine Camera Rigs + Minimal Pause UI

Dynamic camera transitions using Cinemachine. Minimal pause menu (play/resume, exit) for usability.

Tech keywords: Unity C#, Cinemachine, Transform/Quaternion SLERP, time-based input timing, Particle System, simple respawn logic, UI Canvas.
