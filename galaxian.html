Galaxian Remake

C · 2D Sprite Library · Arcade Replica

Classic Gameplay · Intro Cinematics · Scoring System

Role: Solo Developer · Tech: C · Focus: 2D Graphics, Game Logic · Year: 2021

  • Watch on YouTube

Overview

Galaxian Remake is a faithful recreation of the classic arcade shooter. Programmed in C during my first year of coding, it uses a simple 2D sprite library and a global spritesheet for all characters and animations. Despite being an early project, the game was polished to deliver a near-exact replica of the original. I also added a small extra feature not present in the original: the ability to pause the game with the P key.

Key Features

Intro Cinematics & Enemy Showcase

The game opens with cinematic sequences that introduce enemies and explain their unique attack behaviours, staying true to the arcade original.

Convoy Attacks & Score System

Implemented enemy convoy attack patterns along with a scoring system based on accuracy and type of enemy destroyed, recreating the progression and challenge of the original game.

Menu Screen

Menu & Pause Functionality

Includes a menu screen for starting the game. I also added an extra feature not present in the original: the ability to pause the game with the P key.

Technologies Used

  • Language: C
  • Graphics: Simple 2D Sprite Library (University-provided)
  • Assets: Global Spritesheet (original Galaxian sprites)
  • Gameplay: Convoy AI, Score System, Cinematics, Extra Pause Functionality
